# FastAPI-Simple-Chat
Simple Chat Application with FastAPI

# Get started
```shell
pip install -r requirements.txt
uvicorn main:app --reload
```
샘플 코드에 대한 자세한 설명은 [여기](https://blog.naver.com/shino1025/222891742346)에서 확인하실 수 있습니다.
